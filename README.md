# 🛒 Sistema de Gerenciamento de Compras Online

Uma aplicação web full-stack desenvolvida para o controle e gestão eficiente de fluxos de compras. O sistema oferece uma interface dinâmica para análise de dados e um backend robusto com controle rígido de segurança e regras de negócio direto no banco de dados.

## 🚀 Funcionalidades Principais
* **Autenticação e Segurança:** Fluxo completo de login com controle de sessões seguro no backend.
* **Dashboard Interativo:** Consumo de APIs assíncronas para renderização de gráficos dinâmicos, permitindo a análise visual rápida de métricas de compras.
* **Gestão de Dados Relacionais:** Modelagem complexa aplicando lógica de negócios diretamente no banco de dados através de triggers e procedures.
* **Interface Otimizada:** Frontend construído com ferramentas modernas de build para garantir alta performance e carregamento rápido.

## 💻 Tecnologias Utilizadas
**Frontend:**
* React
* Vite
* Bibliotecas de renderização de gráficos (ex: Chart.js / Recharts)

**Backend & Banco de Dados:**
* Node.js com Express
* MySQL (Triggers e queries otimizadas)
